# GO ATTACK GAME

## About

This projet is a simple multiplayer game created using Canvas, WebSocket and WebRTC

## How to run

To run it is necessary to start the whole project, for that run:

In peer-server folder:

```
npm run dev
```

In public:

```
npm start
```

In server:

```
npm start
```
